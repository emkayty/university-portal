# University Portal - Frontend Web Application
# ========================================

This is the Next.js frontend for the University Portal.

## Getting Started

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Environment Variables

```bash
cp .env.example .env.local
```

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- React Query
- React Hook Form
- Zod
- NextAuth.js

## Project Structure

```
src/
├── app/                 # Next.js App Router
├── components/          # React components
├── hooks/             # Custom hooks
├── lib/               # Utilities
├── services/          # API services
├── stores/            # State management
└── types/            # TypeScript types
```
